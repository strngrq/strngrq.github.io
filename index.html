Программирование Minecraft

В Minecraft для Raspberry Pi есть удобное API для программирования на Python. Но этот майнкрафт родственник не десктопного, написанного на Java а мобильной версии (Android, iOS) . Для десктопа есть forge - фреймворк для программирования на Java, но порог вхождения там существенно выше т.к. Java. Но нашелся хороший человек, который реализовал python API для десктопной версии Инструкция по установке. Вот ссылка на исходники на гитхабе .

Краткий туториал для программирования minecraft на питоне https://www.raspberrypi.org/learning/getting-started-with-minecraft-pi/worksheet/ и еще один

Ну а вот ссылка о том, как на Java программировать. Там возможности существенно больше. https://github.com/TheGreyGhost/MinecraftByExample
